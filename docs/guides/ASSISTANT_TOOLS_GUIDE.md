# Assistant 工具调用功能使用指南

**版本**: v1.0  
**日期**: 2026-03-08

---

## 🎯 功能概述

Assistant 现在具备**智能工具调用能力**，可以：

1. **诊断系统问题** - 检查 Ollama 服务、配置文件等
2. **自动修复故障** - 启动服务、修正配置
3. **提供操作指导** - 详细的使用教程和故障排除

---

## 🔧 可用工具

| 工具名称 | 功能 | 触发关键词 |
|---------|------|-----------|
| **check_ollama_health** | 检查 Ollama 服务状态 | "Ollama 状态"、"检查 Ollama"、"Ollama 无法连接" |
| **start_ollama_service** | 启动 Ollama 服务 | "启动 Ollama"、"Ollama 启动" |
| **check_config** | 检查配置文件 | "检查配置"、"配置问题" |
| **get_system_status** | 获取系统整体状态 | "系统状态"、"健康检查" |
| **provide_guide** | 提供操作指南 | "如何"、"怎么"、"指南" |

## 🎨 工作流程

```
用户输入 → 关键词检测 → 触发工具
                ↓
            执行工具
                ↓
            显示结果
                ↓
        AI 分析 + 建议
                ↓
            显示给用户
```

### 详细流程

1. **用户输入消息** - 在 Assistant 页面输入问题
2. **关键词检测** - 系统检测是否包含工具触发关键词
3. **工具执行** - 如果匹配，异步执行相应工具
4. **显示状态** - 显示 "🔧 正在执行: xxx..."
5. **显示结果** - 显示工具执行结果（成功/失败）
6. **AI 分析** - Ollama 基于结果生成建议
7. **最终回复** - 显示 AI 的分析和操作建议

## 🔍 关键词触发规则

### check_ollama_health
触发条件：包含 "ollama" **且** 包含以下之一：
- 中文: "检查"、"状态"、"无法连接"、"连接不上"
- 英文: "check"、"status"、"cannot connect"、"not working"

### start_ollama_service
触发条件：包含 "ollama" **且** 包含以下之一：
- 中文: "启动"、"开启"、"运行"
- 英文: "start"

### check_config
触发条件：包含 "配置" 或 "config" **且** 包含以下之一：
- 中文: "检查"、"问题"、"错误"
- 英文: "check"、"issue"、"error"

### get_system_status
触发条件：包含 "系统" 或 "system" **且** 包含以下之一：
- 中文: "状态"、"健康"、"检查"
- 英文: "status"、"health"、"check"

### provide_guide
触发条件：包含以下任一关键词：
- 中文: "如何"、"怎么"、"指南"、"教程"
- 英文: "how to"、"guide"、"tutorial"、"help me"

## 🛠️ 故障排除

### 问题 1: 工具未触发

**症状**: 输入关键词但没有执行工具

**原因**: 关键词不匹配或缺少必要组合

**解决**:
- 确保包含明确的工具触发关键词
- 例如: "Ollama" + "检查" 而不是单独的 "检查"

---

### 问题 2: 工具执行失败

**症状**: 显示错误消息

**原因**: 
- Ollama 服务未安装
- 配置文件权限问题
- 网络连接问题

**解决**:
- 检查 Ollama 是否已安装: `which ollama`
- 检查配置文件权限
- 查看详细错误消息

---

### 问题 3: AI 回复不相关

**症状**: 工具结果正确但 AI 回复不相关

**原因**: 推理引擎未正确处理工具结果

**解决**:
- 重启 UI
- 检查 Ollama 模型是否正常
- 查看日志: `/tmp/openclaw.log`

- **provide_guide**: 只读，安全

### 自动执行

目前 `start_ollama_service` 会自动执行，未来版本可能添加用户确认步骤。
